# Readme
## Launch a project from cmd
1. `cd project-folder`
1. `git clone git@gitlab.com:olmits/dan-step-project-3.git`
1. `npm install http-server -g`
1. `http-server`
1. open `http://localhost:8080/cards.html` in a browser
